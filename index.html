Base de Dados do Mercantil

Para auxiliar nos estudos e trabalhos práticos da disciplina, disponibilizamos uma base de dados simulada de um mercantil.

Esta base de dados contém informações relevantes para a compreensão e aplicação dos conceitos de sistemas integrados de gestão em um contexto de varejo.
