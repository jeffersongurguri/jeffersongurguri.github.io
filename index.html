    Exercício 1: Classe Pessoa Crie uma classe chamada Pessoa com os seguintes atributos:
    • nome(string)
    • idade(number)
    Adicione um método chamado apresentar():string que retorna uma string formatada com o nome e a idade da pessoa.     Utilize a sintaxe de classe do JS (ES 2015 ou superior).     Exercício 2: Classe Livro Crie uma classe chamada Livro para um Sistema de Biblioteca. A classe Livro deve apresentar os seguintes membros de classe:
    • Atributos
      • titulo: string
      • autor: string
      • disponivel: boolean
    • Métodos
      • emprestar(): undefined - define disponível como false.
      • devolver(): undefined - define disponível como true.
      • exibirDetalhes(): string - retorna uma string com o título, autor e disponibilidade do livro.
         Exercício 3: Classe ContaBancaria Crie uma classe que represente uma conta bancária (ContaBancaria). A classe deve ter os seguintes membros:
    • Atributos
      • saldo: number
      • titular: string
    • Métodos
      • depositar(valor: number): undefined - adiciona o valor ao saldo.
      • sacar(valor: number): undefined
        • - subtrai valor do saldo, se houver saldo suficiente e o argumento valor não pode ser negativo, caso contrário deve lançar uma exceção do tipo RangeError.
      • exibirSaldo(): string - retorna uma string representado o saldo no formato R$ 99,99
         Exercício 4: Classe Retângulo Crie uma classe que represente um retangulo Retangulo para aplicações matemáticas A classe deve apresentar:
    • Atributos
      • largura: number
      • altura: number
    • Métodos
      • calcularArea(): number - retorna a área do retângulo
      • calcularPerimetro(): number - retorna o perímetro do retângulo
         Exercício 5: Classe Animal e Subclasses Crie uma hierárquia de classes com as classes Animal, Cachorro e Gato.
    1. Crie uma classe chamada Animal com os seguintes atributos
      • nome:string
      • especie: string
    2. Adicione um método em Animal chamado emitirSom(): string que retorna uma string genérica ("Som de animal").
    3. Crie subclasses Cachorro e Gato que herdam de Animal
    4. Sobreescreva o método emitirSom() em cada subclasse para retornar o som específico do animal ("Au au" para cachorro, "Miau" para gato)

    Exercício 6: Mouse sobre o elemento Vamos manipular elementos utilizando a API addEventListener()
    1. Crie um elemento <div> com algum conteúdo.
    2. Adicione um ouvinte de evento mouseover ao elemento <div>.
    3. Quando o mouse passar sobre o elemento, altere a cor de fundo do elemento para amarelo.
    4. Adicione um ouvinte de evento mouseout ao elemento <div>
    5. Quando o mouse sair do elemento, restaure a cor de fundo original.
         Exercício 7: Temporizador com callback Vamos criar um timer de cozinha digital.
    1. Crie um botão com o texto "Iniciar temporizador".
    2. Quando o botão for clicado, use a função setTimeout para executar um callback após 5 segundos.
    3. O callback deve exibir um alerta com a mensagem "Temporizador concluído!".
         Exercício 8: Requisição AJAX com callback Vamos realizar uma requisição Web com XHR ou Fetch.
    1. Crie um botão com o texto "Carregar dados".
    2. Quando o botão for clicado, use o objeto XMLHttpRequest ou a Fetch API para fazer uma requisição GET para um arquivo https://microsoftedge.github.io/Demos/json-dummy-data/64KB.json .
    3. Use um callback para processar a resposta da requisição e exibir os dados em um elemento HTML.
