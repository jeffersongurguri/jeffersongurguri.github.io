Disciplinas Técnicas do Curso Técnico Subsequente em Informática para Internet - IFCE

Campus Baturité

O campus Baturité do Instituto Federal de Educação, Ciência e Tecnologia do Ceará (IFCE) oferece o curso Técnico Subsequente em Informática para Internet na modalidade a distância [1-4]. O curso, com duração de 3 semestres ou 1,5 anos, e início de funcionamento previsto para 2023.1, tem como objetivo geral formar um profissional-cidadão técnico de nível médio, capaz de articular teoria à prática, demonstrando conhecimentos, competências, habilidades e atitudes para o desenvolvimento de atividades inerentes aos métodos de desenvolvimento de software para Internet [3-6].

Abaixo estão as disciplinas técnicas obrigatórias que compõem a matriz curricular do curso, divididas por semestre, e suas respectivas ementas [7-10]:

Primeiro Semestre

Informática Básica (UC2-IB) [8]

A ementa da disciplina Informática Básica inclui Introdução à Informática ; Utilização básica do computador e sistemas operacionais ; Digitação ; Internet e aplicações ; e Uso de ferramentas de escritório para elaboração de documentos de texto, apresentações de slides e planilhas eletrônicas [11].

Lógica de Programação (UC4-LP) [8]

Esta disciplina aborda a introdução à lógica de programação ; constantes ; tipos de dados primitivos ; variáveis ; atribuição ; expressões aritméticas e lógicas ; estruturas de decisão ; estruturas de controle ; estruturas de dados homogêneas e heterogêneas (vetores/arrays e matrizes). Inclui também o desenvolvimento de algoritmos e a transcrição de algoritmos para uma linguagem de programação [12].

Desenvolvimento Web I (UC5-DWI) [9]

A ementa abrange a Introdução à Internet e World Wide Web (WWW), conceitos básicos de arquitetura cliente-servidor e serviços da internet . Detalha Hypertext Markup Language (HTML) em sua estrutura, semântica, elementos, boas práticas e websites com múltiplas páginas. Trata de Cascading Style Sheets (CSS) incluindo especificidade, seletores, elementos, CSS Resets e media queries. Aborda a estruturação para a apresentação da informação (box model, posicionamento com floats, grids) e efeitos gráficos e animações (animações, transições e transformações CSS). Por fim, explora o Design da Experiência de Usuário (UX) , cobrindo Princípios de IHC, princípios e elementos da UX, requisitos de usabilidade, acessibilidade na web, padrões de interação e navegação na Web, layout, composição e avaliação de usabilidade [13].

Sistemas Operacionais (UC7-SO) [9]

Esta disciplina introduz os Sistemas Operacionais , sua gerência de processos e gerência de memória . Aborda o Sistema Operacional Windows com procedimentos de instalação, configuração, gerenciamento, gerenciamento de arquivos e instalação de programas. Similarmente, explora o Sistema Operacional GNU/Linux nos procedimentos de instalação, configuração, gerenciamento, gerenciamento de arquivos e instalação de programas [14].

Lógica Matemática (UC3-LM) [7]

A ementa desta disciplina abrange Sistemas de Numeração ; Teoria dos conjuntos ; Noções básicas de lógica proposicional ; Funções ; e Noções básicas de Teoria dos Grafos [10].

Segundo Semestre

Banco de Dados I (UC8-BDI) [9]

A disciplina introduz os Banco de Dados , com foco em Modelagem Conceitual, Lógica e Física . Inclui a instalação e configuração de um SGBD . Aborda conceitos básicos como tabelas, campos, registros, chave primária, chave secundária e chave estrangeira. Abrange a Linguagem SQL Básica (DML e DDL), Normalização , e a conexão do SGBD a sistemas de informação , além da implementação das operações de CRUD (Create, Read, Update, Delete) [15].

Programação Orientada a Objetos (UC9-POO) [9]

Esta disciplina aborda os fundamentos do paradigma orientado a objetos (abstração, encapsulamento, classes, objetos, atributos, métodos e construtores) e conceitos avançados de orientação a objetos (herança, polimorfismo, classes abstratas, interfaces e pacotes). Inclui também a metodologia de desenvolvimento orientada a objetos , a aplicação dos conceitos de orientação a objetos através de uma linguagem de programação apropriada , o uso de Collections (estruturas de dados genéricas) e o tratamento de exceções [16].

Engenharia de Software I (UC10-ESI) [9]

A ementa cobre os Fundamentos de Engenharia de Software , Processos de desenvolvimento de software e Desenvolvimento ágil de software . Aborda a Engenharia de requisitos , Modelagem de Sistemas , Técnicas de Modelagem Orientada a Objetos e a Linguagem de Modelagem Unificada – UML [17].

Desenvolvimento Web II (UC11-DWII) [9]

Esta disciplina trata da linguagem de desenvolvimento back-end ; aplicações Web (Dinâmica, Gerenciamento de sessão, Controle de Cache, Protocolos de comunicação); acesso a banco de dados em sistemas Web (back-end) (Conexão com Sistema de Gerenciamento de Banco de Dados, CRUD e RESTful API). Também aborda a linguagem de desenvolvimento front-end e o ciclo de desenvolvimento versionado (Controle de versões, Ferramentas de versionamento, Branching e tracking, e Correção de erros) [18].

Redes de Computadores (UC13-RC) [10]

A disciplina abrange os Princípios de Comunicação de dados , Topologias , Arquiteturas de redes de computadores , Nível físico , Nível de enlace , Padrões para nível físico e de enlace , Nível de rede , Ligação Inter-Redes , Nível de aplicação e inclui atividade prática em laboratório físico ou virtual [19].

Projeto Integrador I (UC12-PI1) [9]

A disciplina aborda a pesquisa científica , explorando temáticas relacionadas a questões étnico-raciais, história e cultura afro-brasileira e indígena, bem como educação ambiental. Foca no desenvolvimento de projetos de pesquisa para resolução de problemas que envolvam essas temáticas de forma integradora, buscando desenvolver uma cultura científica interdisciplinar e propostas de pesquisa que dialoguem com a tecnologia [20, 21].

Terceiro Semestre

Banco de Dados II (UC15-BDII) [10]

Esta disciplina visa capacitar o aluno a empregar técnicas avançadas em Banco de Dados Relacionais e a utilizar tecnologias não relacionais . Também introduz conceitos de Engenharia e Ciência de Dados [22].

Engenharia de Software II (UC16-ESII) [10]

A ementa inclui Gerência de Projetos de Software , Métricas de processo e projeto de software , Análise e Projeto de Sistemas Web , Documentação, testes e implantação de software e Ambientes de desenvolvimento de software [23].

Desenvolvimento Web III (UC17-DWIII) [10]

Esta disciplina foca em Frameworks WEB , com uma introdução a frameworks para desenvolvimento WEB. Abrange Padrões de software no desenvolvimento WEB e o acesso a banco de dados em sistemas WEB via framework de mapeamento objeto-relacional (ORM) [24].

Segurança da Informação (UC18-SI) [10]

A ementa trata dos Fundamentos de gerenciamento em redes de computadores , Protocolos de gerenciamento de redes , Metodologias para resolução de problemas em redes e Ferramentas para gerenciamento de redes . Também cobre os Fundamentos de segurança da informação , Introdução à criptografia , Firewalls , Detecção e prevenção de invasões e Segurança em redes sem fio [25].

Programação para Dispositivos Móveis (UC20-PDM) [10]

A disciplina aborda o Histórico do desenvolvimento dos aplicativos móveis ; Interfaces gráficas ; Eventos de aplicações ; Aplicações multimídia ; Comunicação ; Persistência de dados ; Mapas e Geolocalização . Inclui o desenvolvimento prático de um sistema para dispositivos móveis [26].

Projeto Integrador II (UC19-PI2) [10]

Esta disciplina envolve a produção de uma proposta de projeto que contemple as análises de requisitos e de sistemas , planejamento, codificação, testes e documentação . Tem como objetivo capacitar o aluno na realização de um projeto multidisciplinar que integre os saberes das diversas disciplinas do curso, buscando articular na prática a proposta através de um projeto-problema [27, 28]. Esta disciplina também inclui 40 horas de prática profissional supervisionada [27].
