Disciplinas Técnicas do Curso Técnico em Informática para Internet Integrado ao Ensino Médio - IFCE

Campus Crato

O campus Crato do Instituto Federal de Educação, Ciência e Tecnologia do Ceará (IFCE) oferece o curso Técnico em Informática para Internet Integrado ao Ensino Médio [2]. Abaixo estão as disciplinas técnicas que compõem a Formação Técnica e Profissional do curso e suas respectivas ementas:

Introdução a Informática [3]

A ementa da disciplina de Introdução a Informática abrange a História e Evolução da Computação; Sistemas de Numeração (Binário, Octal e Hexadecimal); Dispositivos de Entrada, Saída e E/S; Dispositivos de Armazenamento secundário; Arquitetura de Von Neumann; Máquina de Turing; Introdução ao Estudo sobre Sistemas Operacionais (Conceitos, Funções, Kernel e Shell, Interfaces, Inicialização do PC, Sistemas Monoprogramáveis/Monotarefas, Sistemas Multiprogramáveis/Multitarefas, Sistemas com Múltiplos Processadores); Estudo das Memórias (Principais, Secundárias, Alocação e Recuperação de dados, Paginação, Swapping, Thrashing, Máquina Virtual); Escalonamento da CPU; Estudo do Linux (Histórico, Características, Comandos de Gerenciamento de Arquivos e Diretórios); Preparação para a Instalação do Sistema Operacional (Análise, Formatação, Disco de Boot, Partição da Memória); Suite de Escritório (Office) (Edição de Textos, Cálculos, Apresentação, Correio Eletrônico); e Introdução à Internet (Conceito, Histórico, WWW, HTTP) [4-10].

Linguagem de Programação I [3]

A ementa da disciplina de Introdução a Programação, que pode ser associada a Linguagem de Programação I, inclui Algoritmos e lógica de programação estruturada; Conceitos de tradutores, compiladores e interpretadores; Tipos de dados, variáveis e constantes; Operadores (de atribuição, aritméticos, relacionais e lógicos); Estruturas de controle condicional e de repetição; Funções (parâmetros e retornos de funções); Vetores e Matrizes; e Manipulação de Arquivos [11].

Linguagem e Desenvolvimento Web I [3]

Esta disciplina abrange a Introdução ao Desenvolvimento Web; Arcabouços que auxiliam o desenvolvimento de aplicações Web no lado cliente; Desenvolvimento de páginas Web utilizando a linguagem HTML (Estrutura básica, Elementos, Atributos, Parágrafos, Formatação, Links, Imagens, Tabelas, Listas, Semântica, Formulários, Canvas, HTML5 APIs); Formatação de layouts de páginas Web através da criação de arquivos CSS (Seletores, Backgrounds, Texto, Links, Listas, Tabelas, Borda, Margem, Padding, Posicionamento, Alinhamento); e Programação do comportamento de páginas Web utilizando a linguagem JavaScript (Características, Sintaxe, Tipos de dados, Variáveis, Expressões, Comandos, Funções, Objetos, Vetores, Estruturas de controle de fluxo, de repetição e de sequência) [12-15].

Banco de Dados [3]

A ementa de Banco de Dados inclui Histórico dos Bancos de Dados; Sistemas Gerenciadores de Bancos de Dados; Modelos de Banco de Dados; Mercados e Tendências; Modelagem de Dados (Conceitos Básicos, Modelo Entidade-Relacionamento, Modelo Lógico de Dados, Normalização, Mapeamento ER-Relacional); e SQL (Conceitos Básicos, Comandos de criação, manipulação e consulta); e Projeto de Banco de Dados (Características, Arquitetura, Ciclo de vida de um Sistema e suas fases, Ferramenta Case) [16, 17].

Redes de Computadores [18]

Esta disciplina aborda Conceitos e terminologia de redes de computadores; Modelos de redes: OSI e TCP/IP; Protocolos de redes; Camada Física; Meios de transmissão; Camada de Enlace; Equipamentos de interconexão; Redes Ethernet; Camada de Rede; Endereçamento IP; Divisão de redes IP em sub-redes; Camada de Transporte; Camada de Aplicação; Redes sem Fio e Redes Móveis; Segurança de Redes; e Cabeamento Estruturado [19-24].

Linguagem e Desenvolvimento Web II [18]

A disciplina cobre a Visão geral de desenvolvimento para Web; Revisão de HTML e CSS e Introdução a tópicos mais avançados; Linguagem de Programação Javascript; Desenvolvimento de sites responsivos; Geração e gestão de conteúdos web; e Introdução Geral ao Desenvolvimento para Computação usando Cloud Computing [25-30].

Linguagem de Programação II [18]

Esta disciplina trata de Conceitos básicos e introdutórios à linguagem de programação Java; Noções de programação orientada a objeto (Abstração e modelo de dados, Definição de Objeto, Classes, atributos e métodos, Encapsulamento, Níveis de acesso, Herança e polimorfismo, Sobrecarga de construtores e métodos, Interfaces, Controle de Exceções); Interface Gráfica (Introdução à interface gráfica com a biblioteca Swing do Java, Controle de ações); e Persistência com Java e MySQL (Conceitos básicos de SQL com MySQL, Manipulação de banco de dados com JDBC, Desenvolvimento CRUDE) [31-33].

Programação Mobile [18]

A disciplina aborda o Ambiente de desenvolvimento mobile (Framework Expo, Emulador de dispositivo móvel, Ferramentas de desenvolvimento); Linguagem para desenvolvimento mobile (JSX, Componentes de classe e funcionais, Props e State, Funções, Eventos, UseState e useEffect, UseContext e localStorage, ReactNavigation); Consumo de API REST (Requisição HTTP para API, Axios e Async/Await, Tratando a resposta JSON, Passando a resposta para a View); e Gerando Build e publicando App (Gerando APK e IPA, Publicação nas lojas Apple Store e Google Play) [34-36].

Engenharia de Software [18]

Esta disciplina é Introdução à Engenharia de Software e aborda os seguintes tópicos: Introdução à engenharia de software (Desenvolvimento profissional, Ética, Estudos de caso); Processos de software (Modelos de processo, Atividades, Lidando com mudanças, Rational Unified Process); Desenvolvimento ágil de software (Desenvolvimento ágil e dirigido a planos, Extreme Programming, Gerenciamento ágil de projetos, Escalonamento de Métodos ágeis); Engenharia de requisitos (Requisitos funcionais e não funcionais, Documento de requisitos, Especificação, Processos); Gerenciamento de projetos: planejamento (Escopo do software, Análise de risco, Recursos humanos/hardware/software, Reusabilidade); Gerência de projetos: Métricas de software (Processo de gerência, Iniciando um projeto, Medidas e métricas, Estimativa, Análise de riscos, Determinação de prazos, Monitoração e controle); Gerenciamento de qualidade de software (Qualidade, Padrões, Revisões e inspeções); e Técnicas de teste de software (Fundamentos, Fluxo de informação, Projeto de casos de teste, Teste de caixa branca/preta, Particionamento e equivalência, Análise de valor limite, Ferramentas de testes automatizadas) [37-41].

Linguagem e Desenvolvimento Web III [18]

A ementa da disciplina inclui Arquitetura da web (Conceitos, Rede de Servidores, Servidores de Aplicação); Javascript para backend (Javascript no Servidor, Variáveis e constantes, Tipos de dados, Operadores, Entrada e saída de dados, Estruturas de seleção, Estruturas de repetição, Arrays, Formulários, Cookies e sessões, Classes e objetos, Banco de dados, Funções avançadas); NodeJS e Express (Arquitetura do Nodejs, Servidor básico, NPM, Body-parse, Requisição e resposta, Express, Rotas, Middleware); Versionamento e deploy (Git e Github, Heroku, Variáveis de ambiente); e ReactJS (Linguagem JSX, Componentes, State e Props, Eventos e funções) [42-44].

Interface Homem Máquina com Desenvolvimento de Software [45]

A disciplina aborda Conceitos básicos (Interação, interface, affordance, Qualidade em IHC); Fatores humanos (Necessidade do usuário, Motivação, Limitação do usuário); Processos de design de sistemas em IHC (Processo de design e ciclos de vida, Integração das atividades de IHC com engenharia de software, Prototipagem); Avaliação de Interfaces (Experiência de Usuário, Usabilidade, Comunicabilidade, Acessibilidade); e Contextos de sistemas interativos (Web 3.0, Computação móvel, Internet das coisas, Sistemas com Inteligência artificial) [46-48].
