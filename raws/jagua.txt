Bases da Computação
A ementa de Bases da Computação inclui revisão de conceitos matemáticos básicos aplicados à computação; Teoria dos conjuntos; noções de softwares, hardware e internet; princípios fundamentais da lógica; conceitos e fundamentos sobre Sistemas Operacionais (Windows e GNU/Linux); e Ferramentas de escritório (Editores de Texto e Apresentações; Planilhas eletrônicas: estrutura, principais funções, formatações condicionais, filtros e classificações de dados). O objetivo geral é que o estudante conheça os conceitos e princípios fundamentais da informática e da computação, revise tópicos matemáticos essenciais, compreenda a evolução histórica e o funcionamento da arquitetura de computadores e sistemas operacionais, a organização geral dos computadores e seus componentes, e utilize as principais ferramentas e programas de escritório, explorando conceitos e aplicações computacionais do mercado de trabalho. Os objetivos específicos abrangem revisar matemática básica; analisar o histórico e evolução da arquitetura de computadores e SO; identificar características e estrutura do hardware e software; compreender o funcionamento de módulos computacionais, da internet e redes; entender multiprogramação e multiprocessamento; identificar métodos de gerenciamento (memória, CPU, I/O); estudar sistemas operacionais proprietários e abertos; estudar sistemas de arquivo; e conhecer ferramentas de escritório e técnicas de segurança.
Introdução à Lógica de Programação
A disciplina de Introdução à Lógica de Programação tem como ementa Conceitos fundamentais; Algoritmos (Descrição Narrativa, Fluxograma, Pseudocódigo); Estrutura de dados; Sub-rotinas; e Técnicas de Programação. O objetivo geral é que o aluno compreenda a lógica de construção de algoritmos e seja capaz de construir soluções para problemas do mundo real. Os objetivos específicos incluem entender os diferentes tipos de algoritmo, conhecer as instruções básicas, estudar os principais conceitos de construção de algoritmos, familiarizar-se com uma linguagem de programação e elaborar e implementar algoritmos.
Fundamentos Web
A ementa de Fundamentos Web abrange Introdução à Internet; Conceitos básicos de Redes de Computadores; Introdução à World Wide Web (WWW); Introdução à Web 2.0 e 3.0; e Introdução a tecnologias e programação Web. O objetivo geral é que o estudante conheça o histórico do desenvolvimento das tecnologias Web, seja introduzido às três ferramentas basilares da programação Web, e seja capaz de projetar e implementar aplicações Web de baixa e média complexidade, aplicando conceitos e diretivas de desenvolvimento Web. Os objetivos específicos incluem compreender o desenvolvimento da tecnologia Web a partir da Internet; compreender os fundamentos da Internet e tecnologias Web; compreender conceitos básicos de Redes de Computadores; conhecer as ferramentas básicas para o desenvolvimento de aplicações Web; e desenvolver aplicações Web por meio de técnicas de desenvolvimento.
Banco de Dados
A ementa de Banco de Dados inclui Teoria dos Conjuntos; Banco de Dados (Arquitetura, Componentes e Níveis de Abstração); Modelo Relacional (Elementos, Restrições e Diagrama); Linguagem SQL (DDL, DML e Programação); e Projeto de Banco de Dados. O objetivo geral é que o aluno seja capaz de projetar sistemas de banco de dados e manipular o banco de dados utilizando as operações básicas de criação, exclusão, atualização e consulta. Os objetivos específicos são modelar e implementar bancos de dados relacionais a partir de requisitos; traduzir projetos para linguagem SQL; e estruturar e construir requisições de dados usando instruções SQL.
Desenvolvimento Web I
A disciplina de Desenvolvimento Web I aborda Linguagens de marcação; HTML (Estrutura, Semântica, Elementos, Boas práticas, Multi Page Websites); CSS (Especificidade, Seletores, Elementos, CSS Resets, Media Queries); Browser (Ferramentas do desenvolvedor); Posicionamento (Box model, Posicionamento com floats, Grids); Framework Web, CSS responsivo, Grid responsivo; Ciclo de desenvolvimento versionado (Controle de versões, Ferramentas de versionamento, Branching e tracking, Correção de erros); Linguagens front-end Web (Introdução ao DOM, Manipulação do DOM, Eventos); e Persistência de dados com servidores back-end (TCP/IP, HTTP, HTTP Requests, API’s Web, CRUD e RESTful API). O objetivo geral é que o estudante projete e construa páginas Web interativas, observando boas práticas e integrando elementos com serviços externos usando linguagens de programação Web. Os objetivos específicos são conhecer e identificar elementos de construção de interfaces Web; construir interfaces Web usando modelos e métodos da indústria; ser exposto a novidades na área; e analisar e atender requisitos de usuário.
Introdução à Arquitetura de Redes e Servidores Web
A ementa de Introdução à Arquitetura de Redes e Servidores Web abrange Introdução a Redes (URL, DNS, IP); Arquitetura cliente-servidor; Modelos OSI; Padronização; TCP/IP (Camada de Aplicação, Camada de Transporte, Camada de Rede, Camada Física); Servidores WEB (Administração de servidores Linux, TCP/IP, DHCP, Acesso Remoto, Serviço WEB); e Cloud Computing. O objetivo geral é que o estudante compreenda o funcionamento da Internet, desde protocolos e arquiteturas de redes, até noções introdutórias de administração e implementação de serviços de rede em sistemas operacionais livres. Os objetivos específicos são conhecer os princípios de redes de computadores; diferenciar arquiteturas de redes; conhecer os principais protocolos da pilha TCP/IP; implantar serviço WEB; e implantar serviço de DHCP.
Programação Orientada a Objetos e Análise de Projeto de Sistemas Orientado a Objetos
A ementa dessa disciplina compreende Visão geral do paradigma orientado a objetos; Classes, métodos, encapsulamento, herança, polimorfismo aplicados a Java e JavaScript; Collections; Tratamento de exceções; Interface gráfica; Geração de código executável; Análise e modelagem orientada a objetos; Projeto orientado a objetos; Aplicação de ferramentas computacionais de apoio ao processo de análise e projeto de sistemas; e Desenvolvimento da análise e projeto baseado em objetos de um sistema. O objetivo geral é que o aluno compreenda o mundo real e utilize a abstração para mapeá-lo em classes e objetos, construindo programas que solucionem problemas variados, além de interpretar as necessidades do usuário e mapeá-las para diagramas UML que servirão como insumo para o projeto de sistemas orientado a objetos. Os objetivos específicos incluem compreender o paradigma de Programação Orientada a Objetos; diferenciar programação estruturada de POO; elaborar e implementar algoritmos utilizando POO; aplicar boas práticas do paradigma; conhecer e utilizar UML como linguagem de modelagem; desenvolver diagramas UML para as fases de análise, projeto e implementação; compreender ferramentas computacionais para análise de sistemas; e obter noções básicas de engenharia de requisitos.
Produção de Conteúdo Técnico
A ementa desta disciplina aborda Comunicação Oficial; Linguagem formal e técnica; e Estudo e produção de textos técnico-científicos, tais como: relatório, resumo, resenha, artigo, projeto e seminário. O objetivo geral é que o aluno reconheça e produza textos de forma coerente, analisando, interpretando e aplicando os recursos de linguagens, e relacionando textos com seus contextos, considerando a natureza, função, organização e estruturas específicas da área técnica. Os objetivos específicos incluem reconhecer a importância da leitura para a produção de textos técnicos; reconhecer a importância das linguagens verbal e não verbal para resolução de problemas técnicos; identificar elementos que contribuem para a progressão temática e estruturação de textos; ampliar o vocabulário e a capacidade de pensamento objetivo; estimular a expressão oral, a leitura pública e o debate de ideias; identificar, compreender e utilizar as convenções da língua nos gêneros técnicos; e ler e produzir textos orais e escritos na área técnica, considerando as condições discursivas de produção.
Desenvolvimento Web II
A ementa de Desenvolvimento Web II cobre Aplicação Web (Dinâmica, Gerenciamento de sessão, Controle de Cache, Protocolos de comunicação); Linguagem de desenvolvimento back-end (Introdução); Frameworks Web (Introdução a frameworks para desenvolvimento web); Acesso a banco de dados em sistemas Web (Conexão com Sistema de Gerenciamento de Banco de Dados); e Engenharia Web (Desempenho com cluster, Balanceamento de carga, alta disponibilidade, criptografia). O objetivo geral é projetar e construir sistemas Web completos, observando boas práticas e aprendendo sobre a integração com serviços externos, balanceamento de carga e vulnerabilidades. Os objetivos específicos são conhecer e identificar elementos da construção de sistemas Web; construir sistemas Web usando modelos e métodos da indústria; ser exposto a novidades na área para progressão profissional; e analisar e atender requisitos de usuário para encaixe no mercado de trabalho.
Engenharia de Software e Gestão de Projetos
A ementa de Engenharia de Software e Gestão de Projetos inclui Definição de Engenharia de Software; Processos de desenvolvimento de software; Desenvolvimento ágil de software; Engenharia de Requisitos; Testes e Qualidade de software; e Gestão de Projetos. O objetivo geral é entender os conceitos relacionados ao surgimento e à evolução da Engenharia de Software, suas técnicas e metodologias, e sua aplicação e importância em diversas áreas de desenvolvimento de software. Os objetivos específicos são conhecer as fases do processo de desenvolvimento de software; aprender e aplicar os principais modelos de processo; conhecer e aplicar metodologias de testes e qualidade de software; e entender os principais conceitos de gerenciamento de projetos.
Prática Profissional
A ementa da disciplina de Prática Profissional abrange Projeto de Vida; Levantamento e Análise de Requisitos; Projeto de Software; Implementação; Teste; Integração; e Evolução. O objetivo geral é projetar e construir sistemas capazes de resolver problemáticas do cotidiano, respeitando o ciclo de vida de um software completo, desde a concepção da solução até a implantação do projeto. Os objetivos específicos são analisar os requisitos do sistema a ser implementado; projetar o sistema de acordo com um modelo de ciclo de vida; projetar o código a ser implementado; realizar testes de unidade, integração e aceitação; e realizar a operação e manutenção do sistema proposto.
Programação para Dispositivos Móveis
A ementa de Programação para Dispositivos Móveis inclui Histórico do desenvolvimento dos aplicativos móveis; Interfaces gráficas; Eventos de aplicações; Aplicações multimídia; Comunicação; Persistência de dados; Mapas e Geolocalização; Desenvolvimento Prático de um Sistema para Dispositivos Móveis; Fundamentos de teste de software; e Ferramentas e estratégias de testes. O objetivo geral é aprender a construir os mais variados aplicativos para dispositivos móveis, com foco em smartphones e tablets, exercitando conceitos de Programação Orientada a Objetos e desenvolvimento ágil, e planejando e desenvolvendo casos de testes para as diversas fases do desenvolvimento do aplicativo. Os objetivos específicos são analisar e identificar características de potenciais aplicativos; ser capaz de arquitetar e implementar aplicações para dispositivos móveis; e aplicar técnicas de desenvolvimento de softwares em dispositivos móveis.
